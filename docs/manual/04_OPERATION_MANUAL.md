# 管理者向け操作

## ログイン

1. ブラウザで `https://[ドメイン]/admin/login` にアクセス
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

```
初期管理者アカウント:
メール: admin@example.com
パスワード: password（要変更）
```

---

## ダッシュボード

ログイン後、ダッシュボードが表示されます。

### 表示内容
- 📊 総イベント数
- 👥 総受験者数
- 📝 総問題数
- ⏰ 開催中イベント

### サイドメニュー
| メニュー | 機能 |
|----------|------|
| ダッシュボード | 概要表示 |
| イベント管理 | 試験イベントの作成・編集 |
| 問題管理 | 問題の作成・編集 |
| 練習問題管理 | 練習問題の作成・編集 |
| 成績管理 | 試験結果の確認 |
| 統計・グラフ | 成績分析 |
| アカウント管理 | ユーザー・管理者管理 |

---

## イベント管理

### イベント作成

1. サイドメニューから「イベント管理」をクリック
2. 「新規イベント作成」ボタンをクリック
3. 以下の情報を入力：

| 項目 | 説明 | 例 |
|------|------|-----|
| イベント名 | 試験の名称 | 2026年度春季採用試験 |
| パスフレーズ | 受験時に入力するコード | SPRING2026 |
| 開始日時 | 試験開始日時 | 2026-04-01 09:00 |
| 終了日時 | 試験終了日時 | 2026-04-01 18:00 |

4. 「作成」ボタンをクリック

### 問題の割り当て

1. イベント一覧から対象イベントの「編集」をクリック
2. 「問題を追加」セクションで問題を選択
3. ドラッグ&ドロップで出題順を調整
4. 「保存」をクリック

### イベントの状態

| 状態 | 色 | 説明 |
|------|-----|------|
| 開催前 | 青 | 開始日時より前 |
| 開催中 | 緑 | 開始〜終了の間 |
| 終了 | グレー | 終了日時を過ぎた |

---

## 問題管理

### 問題作成

1. サイドメニューから「問題管理」をクリック
2. 「新規問題作成」ボタンをクリック
3. 以下の情報を入力：

| 項目 | 説明 |
|------|------|
| 問題文 | 問題の本文 |
| ジャンル | アルゴリズム / データ構造 / ネットワーク など |
| 難易度 | 1〜5（5が最難） |
| 配点 | 正解時の得点 |
| 選択肢 | 4つの選択肢（1つを正解に設定） |

4. 「作成」ボタンをクリック

### 問題の編集

1. 問題一覧から「編集」をクリック
2. 内容を修正
3. 「保存」をクリック

### 問題のインポート

1. 「インポート」ボタンをクリック
2. CSVファイルを選択
3. プレビューを確認
4. 「インポート実行」をクリック

---

## 練習問題管理

### 練習問題の作成

1. サイドメニューから「練習問題管理」をクリック
2. 「新規作成」ボタンをクリック
3. 問題情報と**解説文**を入力
4. 「作成」をクリック

> **本試験との違い**: 練習問題には解説文があり、回答後に表示されます

---

## 成績管理

### 全体成績一覧

1. サイドメニューから「成績管理」をクリック
2. 全受験者の成績一覧が表示されます

### イベント別成績

1. 「イベント管理」から対象イベントをクリック
2. 「成績一覧」をクリック

### 表示項目
- 受験者数
- 平均スコア
- ページ数（ナビゲーション用）

### 操作
- 検索: ユーザー名、メール、ランクで絞り込み
- ソート: 列ヘッダーをクリックで並び替え
- 詳細: 行をクリックで詳細表示

### 個人詳細

- 各問題の正誤
- 回答時間
- 総合得点とランク

---

## 統計・グラフ

### 表示内容

1. **ランク分布**: Platinum / Gold / Silver / Bronze の人数
2. **スコア分布**: ヒストグラム
3. **ジャンル別正答率**: レーダーチャート
4. **難易度別正答率**: 棒グラフ

### フィルター

- イベント選択
- 期間指定
- ランク絞り込み

---

## アカウント管理

### ユーザー一覧

1. サイドメニューから「アカウント管理」をクリック
2. 「ユーザー」タブを選択

### ユーザー作成（事前登録）

1. 「新規ユーザー作成」をクリック
2. 名前、メール、パスワードを入力
3. 「作成」をクリック

### 管理者一覧

1. 「管理者」タブを選択

### 管理者作成

1. 「新規管理者作成」をクリック
2. 名前、メール、パスワード、権限を入力
3. 「作成」をクリック

### 権限レベル

| 権限 | 説明 |
|------|------|
| super_admin | 全権限（管理者作成可） |
| admin | 通常管理（管理者作成不可） |

# 受験者向け操作

## ログイン

1. `https://[ドメイン]/login` にアクセス
2. メールアドレスとパスワードを入力
3. 「ログイン」をクリック

## 試験を受ける

1. ダッシュボードで開催中の試験を確認
2. 「試験を開始」をクリック
3. パスフレーズを入力
4. 「開始」をクリック

## 試験中の操作

### 回答方法
1. 問題文を読む
2. 選択肢から1つを選択
3. 「次の問題へ」をクリック

### 画面表示
- 上部: 進捗バー（何問目/全問数）
- 中央: 問題文と選択肢
- 下部: ナビゲーションボタン

### 制限事項
- タブ切り替えで警告表示
- 制限時間（イベントによる）
- 一度回答すると戻れない

## 結果確認

### 試験直後
- 総合得点
- ランク（Platinum/Gold/Silver/Bronze）
- 各問題の正誤

### 過去の結果
1. 「マイページ」をクリック
2. 「成績履歴」を選択
3. 過去の試験結果を確認

# ゲスト向け操作

## トップページ

`https://[ドメイン]/` にアクセス

### 表示内容
- システム紹介
- 練習問題へのリンク
- ゲスト試験開始

## 練習問題

### 開始方法
1. 「練習問題」をクリック
2. 問題カテゴリを選択
3. 「開始」をクリック

### 練習中の操作
1. 問題を読む
2. 回答を選択
3. 「回答する」をクリック
4. **解説を確認**
5. 「次の問題へ」

> 練習問題では回答後に解説が表示されます

## ゲスト試験

### 開始方法
1. 「ゲスト試験」をクリック
2. 名前を入力
3. パスフレーズを入力
4. 「試験開始」をクリック

### 注意事項
- ゲストアカウントは一時的
- 結果は保存されますが、後から確認する場合は
  セッションUUIDが必要です

# よくある質問（FAQ）

## 管理者向け

**Q: パスフレーズを忘れた場合は？**
A: イベント管理画面から確認できます。

**Q: 問題を一括登録したい**
A: CSVインポート機能を使用してください。

**Q: 特定の受験者の結果を削除したい**
A: 成績管理から該当セッションを選択し、削除ボタンをクリック。

## 受験者向け

**Q: 試験中にブラウザを閉じてしまった**
A: 再度ログインして試験画面に戻ると、途中から再開できます。

**Q: 結果が表示されない**
A: 管理者に連絡して、イベント設定を確認してもらってください。

## ゲスト向け

**Q: 練習問題は何回でもできる？**
A: はい、何度でも練習できます。

**Q: ゲスト試験の結果を後から見たい**
A: 結果画面に表示されるセッションIDを控えておいてください。

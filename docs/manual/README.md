## クイックインストール

```bash
# 1. プロジェクトを展開
unzip ProgrammerAptitudeTest.zip
cd ProgrammerAptitudeTest

# 2. 環境設定ファイルをコピー
cp .env.example .env

# 3. インストールスクリプトを実行
chmod +x install.sh
./install.sh
```

## 1. 一般ユーザー向け操作

### 1.1 新規登録
1. トップページ（https://aws-sample-minmi.click）にアクセス
2. 「新規登録」ボタンをクリック
3. 以下の情報を入力:
   - 名前（氏名）
   - メールアドレス
   - パスワード（8文字以上）
   - パスワード確認
4. 「登録」ボタンをクリック

### 1.2 ログイン
1. トップページにアクセス
2. 「ログイン」ボタンをクリック
3. メールアドレスとパスワードを入力
4. 「ログイン」ボタンをクリック

### 1.3 セッションコード入力
1. ログイン後、セッションコード入力画面が表示される
2. 試験官から提供されたセッションコードを入力
3. 「参加」ボタンをクリック

### 1.4 練習問題の受験
1. 第1部の練習問題が表示される
2. 各問題の選択肢（A〜E）から回答を選択
3. 制限時間内に全問回答
4. 「完了」ボタンをクリック
5. 解説画面で正解と解説を確認
6. 「本番試験へ」ボタンをクリック

### 1.5 本番試験の受験
1. 試験説明画面を確認し「開始」をクリック
2. 各パートの問題に回答:
   - 第1部: 計算問題（40問）
   - 第2部: 図形問題（30問）
   - 第3部: 命令表問題（25問）
3. 各パート終了後、次のパートの練習問題へ
4. 全パート完了後、結果画面が表示される

### 1.6 結果確認
- 結果画面で以下を確認可能:
  - 各パートの正解数・不正解数・未回答数
  - 総合スコア
  - ランク（Platinum/Gold/Silver/Bronze）
- 「認定証を表示」で認定証を取得可能

## 2. ゲストユーザー向け操作

### 2.1 ゲスト受験開始
1. トップページで「ゲスト」ボタンをクリック
2. セッションコードを入力
3. ゲスト情報入力画面で:
   - 学校名を入力
   - 氏名を入力
4. 「試験を開始」ボタンをクリック

### 2.2 練習問題・本番試験
- 一般ユーザーと同様の流れで受験
- 結果はデータベースには保存されない（セッション終了で消去）

## 3. 管理者向け操作

### 3.1 管理者ログイン
1. https://aws-sample-minmi.click/admin/login にアクセス
2. 管理者メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

### 3.2 ダッシュボード
- ログイン後、ダッシュボードが表示される
- 左側メニューから各機能にアクセス可能

### 3.3 イベント管理
1. 「イベント管理」をクリック
2. 「新規作成」で新しいイベントを作成:
   - イベント名
   - セッションコード（パスフレーズ）
   - 開始日時・終了日時
   - 試験タイプ（full/45min/30min）
   - 各パートの時間・問題数設定
3. 作成済みイベントの編集・削除が可能

### 3.4 結果管理
1. 「結果管理」をクリック
2. イベント別に受験者の結果を確認
3. CSVエクスポートが可能

### 3.5 ユーザー管理
1. 「ユーザー管理」をクリック
2. 登録ユーザー一覧を表示
3. ユーザーの詳細確認・削除が可能

### 3.6 管理者アカウント管理
1. 「管理者管理」をクリック
2. 管理者アカウントの一覧表示
3. 「新規管理者登録」で新しい管理者を追加:
   - 名前
   - メールアドレス
   - パスワード
4. 既存管理者の削除が可能

### 3.7 問題管理
1. 「問題管理」をクリック
2. 本番問題・練習問題の一覧表示
3. 問題の追加・編集・削除が可能

## トラブルシューティング

### ログインできない
- パスワードを忘れた場合は「パスワードをお忘れですか？」から再設定
- メールが届かない場合は迷惑メールフォルダを確認

### セッションコードが無効
- セッションコードの有効期限を確認
- 大文字・小文字を正確に入力

### 試験中に接続が切れた
- 回答は自動保存されているため、再ログインで続きから再開可能
- ゲストの場合はセッションが切れると最初からやり直し

### 画面が正しく表示されない
- ブラウザのキャッシュをクリア
- 推奨ブラウザ: Chrome, Firefox, Safari (最新版)
